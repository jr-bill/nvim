# nvim

### Requirements

- python3
- pynvim (pip install pynvim)
- install node.js
    > curl -sL install-node.now.sh | sh
- Coc installed with:
    - coc-css
    - coc-emmet
    - coc-html
    - coc-json
    - coc-phpls
    - coc-snippets
    - coc-vimlsp
    - coc-tsserver
    - coc-explorer
- install intelephense with npm
    - npm -g i intelephense
